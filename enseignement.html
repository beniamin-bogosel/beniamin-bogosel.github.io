Computational Maths 2: Numerical Optimization (2018-present)

This course takes place in the second year of the Bachelor program at Ecole Polytechnique. It deals with optimization problems with an emphasis on numerical algorithms.

Course Materials:

  • Optimization in dimension one: Courses 1-2
    • Python Notebook illustrating the trisection method: Trisection
    • Python Notebook: Newton method
    • Python Notebook: Line-search
  • Optimization in higher dimensions (Gradient Descent, Newton): Courses 3-4
    • Python Notebook: Gradient Descent
    • Python Notebook: Ill conditioned case
    • Python Notebook: Newton method
  • Optimization in higher dimensions (quasi-Newton, Conjugate Gradient,...): Course 5
    • Python Notebook: Scipy Minimize Example
  • Optimization under constraints (theory of Lagrange multipliers, intro to Linear Programming): Courses 6-7
    • Python Notebook: Example using scipy.linprog
    • Python Notebook: Optimal Assignment Problem
    • Matlab code: Solving Sudoku with LP sudokuLP.m
